## Описание модулей

> В папке [src](./src) лежат файлы с реализацией задачи, заданной в лабораторной работе.
> В файле [main.py](./src/main.py) описана точка входа в приложение - функция **main**.Также этот файл содержит минимально необходимую логику парсера, необходимого для интерактивного взаимодействия пользователя с различными модулями проекта. Интерактивный режим ожидает верный синтаксис ввода (описан ниже), иначе поведение не определено.
> В файле [functions.py](./src/functions.py) реализованы функции для рассчета чисел Фиббоначи и факториала.
> В файле [sorting.py](./src/sorting.py) реализован класс **Sortings**, которые содержит различные методы сорнировки списков.
> В файле [structures.py](./src/structures.py) реализованы классы **Stack** и **Queue**, которые представляют собой реализацию стека и очереди на основе list.
> В папке [tests](./tests) лежат unit-тесты на различные модули.
> В файле [test_functions.py](./src/test_functions.py) лежат тесты на модуль [functions.py](./src/functions.py)
> В файле [test_structures.py](./src/test_structures.py) лежат тесты на модуль [structures.py](./src/structures.py)
> В файлах [test_sortings.py](./src/test_sortings.py) и [test_sortings_by_generators.py](./src/test_sortings_by_generators.py) лежат тесты на модуль [sorting.py](./src/sorting.py). При этом сами генераторы тестов реализованы в файле [testcase_generation.py](./src/testcase_generation.py)
> В файле [benchmark_tests.py](./src/benchmark_tests.py) содержится алгоритм сравнения эффективности реализованных сортировок через бенчмарки.

## Инструкции по запуску модулей и тестов

> Запуск интерактивной работы со структурами:
```
python -m main
```
> Синтаксис для взаимодействия со структурами и функциями:

1. Создание структуры (Stack/Queue) :
```
type name value(optional)
```
Например,
```
stack a [1,2,3]
queue b
```
Ограничения:
- Value (список) вводится без пробелов между символами.
- В интерактивном режиме поддерживается только тип int для элементов списка (согласно ТЗ).
- Структуры с повторяющимися именами перезаписывают более старые.

2. Вызов методов структур:
```
name.method(args)
```
Например,
```
a.push(3)
b.print()
b.len()
```

3. Запуск алгоритмов сортировки:
```
algo_name list_value
```
Например,
```
bubble_sort [1,5,2,3]
```
Ограничения:
- Интерактивный ввод подразумевает непустой массив
- Типы введеных данных должны соответствовать алгоритму сортировки

4. Запуск функций из файла [functions.py](./src/functions.py):
```
func_name arg
```
Например,
```
factorial 10
```
Ограничения:
- Соответствие ввода тому, что ожидает функция

5. Запуск тестов:
```
cd tests
pytest test_sortings.py -v
pytest test_structures.py -v
pytest test_sortings_by_generators.py -v
pytest test_functions.py -v
python benchmark_tests.py
```
